Introduction

The initial version was published at the alt.sources newsgroup in 1991, and version 1.0 was released in 1994.

Variables

What is a Variable in Python?

A Python variable is a reserved memory location to store values. In other words, a variable in a python program gives data to the computer for processing.
Every value in Python has a Data Type. Different data types in Python are Numbers, List, Tuple, Strings, Dictionary, etc. Variables can be declared by any name or even alphabets like a, aa, abc, etc.

How to Declare and use a Variables

Lets see an example, we will declare a variable called "foo" and print it.

foo = 42
print(foo)

As we can see when we declare foo = 42 we are assigning to our foo variable a number, so this way our variable's Data Type converts to an Integer.

We can also declare multiples variables in a single code line and assign it a value for each of them.

a, b, c = "42", 3, [1,2,3]
print(a)
print(b)
print(c)

output
"42"
3
[1,2,3]

Data Types

Python is a non-typed programming language, that means when we declare our variables we don't have to add the type of that variable.

At first this could be a bit confused but when we learn how every data type is represented we'll notice how It can be readable and understood.

At the moment that we assign a value to our variables, it would represent a data type.
Lets see some examples:

#Declaring and assigning a Integer value
myInteger = 42

#We could also reassign a value for that same variable (It can be different data type )
myInteger = "Laptop"

# Now myInteger variable has been reassigning to the value of "Laptop" changing his type to String

# String variables
myString = "Hello World"

# List variables
myList = [1, 2, 3, ['h','e','llo'], "Home"]

# Tuple variables
myTuple = (4, 3)

# Set variables
mySet = {1, 2, 3, 4, 42}

Conditional Sentences

Python supports the usual logical conditions from mathematics:

  • Equals: a == b
  • Not Equals: a != b
  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than:a > b
  • Greater than or equal to:a >= b

These conditions can be used in several ways, most commonly in "if statements" and loops.
An "if statement" is written by using the if keyword.

a = 5
b = 10

#if condition is true then we execute the code inside unless go to else code.
if a > b:
print(f'{a} is greater than {b}')
else:
print(f'{a} is less than {b}')

Output:
5 is less than 10

Repetitive Sentences

Python has two primitives loops commands

  • while loops
  • for loops

While loop

With while loop we can execute the code inside as long as condition remains true

i = 10
while i > 10:
print(i)
i-= 1

# Output:
10
9
8
...
5
...
2
1

The while loop requires relevant variables to be ready, in this example we need to define an i, i, which we set to 1.

For loops

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).
This is less like the for keyword in other programming languages, and works more like an iterator method as found in other object-orientated programming languages.
With the for loop we can execute a set of statements, once for each item in a list, tuple, set etc.

colors = ["Yellow", "Blue", "Red", "White"]
for c in colors:
print(c)

# Output
"Yellow"
"Blue"
"Red"
"White"

The for loop does not require an indexing variable to set beforehand.
